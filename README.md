## 연락 방법

제 디스코드 사용자명 `howeverina`를 통해 아래 양식으로 연락주시면, 서브도메인에 `CNAME` 또는 `A` 레코드를 넣어드려요. 현재는 1차지인만 받고 있어요.

### 요청 양식

- **사용 중인 닉네임**: 1차지인 식별용이에요.
- **서브도메인**: 영문을 추천드려요. `sungsimdang` 같은 기존 유명 업체 이름은 거부될 수 있어요.
- **용도**: 프로젝트를 간략히 소개해주세요. 대전을 거점으로 하거나 연관이 있는 프로젝트가 좋을 것 같아요. (사이트 운영자의 거주지와는 무관해요!)
- **A/CNAME 레코드**: 아이피를 주시면 `A`에 넣어드리고, 도메인을 주시면 `CNAME`에 넣어드려요.

### 말도 안 되는 요청의 경우...

- 시간이 맞는 경우 여러가지 인증용 TXT 레코드 등...을 넣어드릴 수 있어요. 디스코드로 연락 주세요.

- 최초 등록 수고비 1000원을 받고 있어요.
